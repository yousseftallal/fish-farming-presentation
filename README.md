# Smart Fish Farming Presentation

This is an interactive HTML5 presentation for the "Smart Fish Farming" graduation project.

## Project Overview
A smart management system for aquaculture using IoT, AI, and Automation to improve fish production efficiency and sustainability.

## Features
- **3D Passive Transitions**: Clean, professional slide transitions.
- **Responsive Design**: Works on Desktop and Mobile.
- **Deep Ocean Theme**: Professional visual aesthetic.

## Tech Stack
- HTML5 / CSS3
- JavaScript (Vanilla + GSAP for animations)
- GitHub Pages for hosting

## Content
1. Team Introduction
2. Table of Contents
3. Introduction to Aquaculture
4. Motivation
5. Business Capabilities
6. Technical Analysis (IoT/AI)
7. Sensor Demo
8. UI Design
